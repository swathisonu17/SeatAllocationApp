#  Exam Hall Seat Allocation System

A Streamlit-based web application designed to automate the process of **exam hall seat allotment** efficiently and accurately.

This project helps colleges reduce manual work and ensures proper seat distribution based on available benches, halls, and student data.

---

##  Features

✅ Upload student data using CSV  
✅ Automatic seat allocation based on hall capacity  
✅ Generates allotment plan instantly  
✅ View student-wise and hall-wise seating arrangement  
✅ Export reports in PDF/CSV format  
✅ User-friendly dashboard interface  

---

##  Tech Stack

- **Frontend + Backend:** Streamlit (Python)  
- **Data Handling:** Pandas  
- **Report Generation:** ReportLab  
- **File Storage:** CSV-based system  
- **Deployment Ready:** Streamlit Cloud / Render
